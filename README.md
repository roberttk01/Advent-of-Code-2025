# Advent of Code 2025

Personal solutions for [Advent of Code 2025](https://adventofcode.com/2025) puzzles, written in Python.

## Structure

```
Day N/
├── input                              # Puzzle input
├── AoC_DayN_Part1.py                  # Part 1 solution
└── AoC_DayN_Part2.py                  # Part 2 solution
```

## Progress

| Day | Part 1 | Part 2 |
|-----|--------|--------|
| 1   | ⭐     | ⭐     |
| 2   | ⭐     | ⭐     |
| 3   | ⭐     | ⭐     |

## Running Solutions

```bash
cd "Day N"
python AoC_DayN_Part1.py
python AoC_DayN_Part2.py
```

Each solution reads from the `input` file in its directory.
